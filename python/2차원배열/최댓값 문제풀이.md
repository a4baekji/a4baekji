문제에 주어진 배열을 그대로 사용했었으나 문제에선 행렬을 직접 입력 받는 것을 요구했었어서 처음에 런타임 에러가 떴으며, 최댓값을 저장하는 변수 명을 max로 써서 함수 이름과 겹쳐 런타임 에러가 떴었어서 모두 수정하였다.
그 과정에서 max_val의 값을 a[0][0]으로 설정해서 입력 받는 배열의 첫번째 값을 최댓값으로 설정하고 코드를 실행했는데 런타임 에러가 발생했다.
코드를 다시 보니 if문에 a[i][j] > max_val: 로 적어놨는데 이 경우 첫 번째 값이 최대일 때 모든 배열의 값과 값 비교는 하나 a[i][j] 값이 max_val 값보다 큰 경우가 없게 되어
if문 안의 인덱스가 갱신되지 않는다는 문제가 있다.
따라서 초기 인덱스 값을 1, 1로 초기화 하거나 if문의 조건을 >=로 바꿔주어 문제를 해결했다.
